## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start the development server**:
    ```bash
    npm run dev
    ```

3.  **Build for production**:
    ```bash
    npm run build
    ```

## Design Philosophy

This project follows a "less is more" philosophy:
- **Restraint**: Quiet confidence over loud calls to action.
- **Editorial**: Layouts inspired by high-end fashion and lifestyle magazines.
- **Atmosphere**: Every interaction is slow, deliberate, and smooth.
